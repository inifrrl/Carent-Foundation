## 🚀 QUICK START - Mulai Di Sini!

### 📤 Upload ke GitHub & 🌐 Deploy (20 menit)

```bash
# 1. Buat repository di github.com dulu
# 2. Jalankan commands ini:

git init
git add .
git commit -m "Initial commit: Carent Indonesia website"
git remote add origin https://github.com/USERNAME/carent-indonesia-website.git
git branch -M main
git push -u origin main

# 3. Buka vercel.com → Import repository → Deploy
# 4. Done! Website live! 🎉
```

## 🎯 Cara Upload & Deploy

### Pilihan 1: Manual (Recommended)
1. Baca [`QUICK_START.md`](./QUICK_START.md)
2. Follow step-by-step
3. Website live!

### Pilihan 2: Script Otomatis

**Mac/Linux:**
```bash
chmod +x upload-to-github.sh
./upload-to-github.sh
```

**Windows:**
```
Double-click: upload-to-github.bat
```

### Pilihan 3: Vercel Deploy
1. Push ke GitHub
2. Vercel.com → Import repository
3. Deploy (otomatis!)
